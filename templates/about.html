About AdOptimizer

AdOptimizer is a lightweight, machine learning–powered web app built using Flask, MySQL, and Python. It helps marketers analyze ad performance, predict campaign results, and optimize ad budgets efficiently.

Core Features

Real-time Campaign Tracking

Monitor and track campaign performance directly from the database for instant insights.

ML-based Predictions

Leverage machine learning for accurate CTR and conversion predictions based on your data.

Automated Budget Recommendations

Get intelligent suggestions to allocate your ad spend more effectively and maximize ROI.

Simple, Modern UI

A user-friendly interface built with Flask templates for easy navigation and management.
